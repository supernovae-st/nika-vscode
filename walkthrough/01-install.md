# Install the nika engine

The extension drives the `nika` binary · one file, no Docker, no Python.

```bash
brew install supernovae-st/tap/nika     # macOS / Linux
# or
curl -fsSL https://nika.sh/install.sh | sh
```

The extension can also **download it for you** · it will offer when no binary
is found (`nika.server.autoDownload`).
